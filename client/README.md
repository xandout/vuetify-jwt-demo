# vuetify-jwt-demo

## Set it up yourself

```bash
npm -i -g @vue/cli
vue create . #create new vue app in PWD
vue add router
vue add vuetify
```

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
